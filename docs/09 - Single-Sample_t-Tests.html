3  One-Sample t-tests

Sample Problems

4 About

the one-sample t-Test problems use the RamdomData class which requires:

  • the groups variable set to 1: groups = 1
  • an intger for the sample size (e.g., n = 30)
  • a call to the one_sample_t_test() method

In these sample problems, the per-group sample size is randomly set between 5 and 15. An example funciton call is included below

sample_size = random.randint(5,15)
RandomData(groups = 1, n = sample_size).one_sample_t_test()

5 Problem 1

sample_size = random.randint(5,15)
RandomData(groups = 1, n = sample_size).one_sample_t_test()

Given the following data, is the mean of \(Group_A\) significantly different from \({13}\)? Use a \({2}\) tailed-test with \(\alpha = {0.05}\)

A
19
19
20
19
18
19
19
20


The necessary summary statistics for these data
\[M_A = {19.12}\] \[s^2 = {0.41}\] \[n = {8}\]

State the Hypotheses
\[H_0: \mu = 13\] \[H_1: \mu \ne 13\]


The decision criteria:

\(t_{crit} = \pm{2.36}, \alpha_{two-tailed} = {0.05}, df = {7}\)

Calculate the standard error
\[s_M = \sqrt{\frac{s^2}{n}}\]
\[s_M = \sqrt{\frac{0.41}{8}}\]
\[s_M = \sqrt{0.05}\]
\[s_M = {0.22}\]

calculate \(t_{obt}\)
\[t_{obt} = {\frac{M - \mu}{s_M}}\]
\[t_{obt} = \frac{19.12 - 13}{0.22}\]
\[t_{obt} = \frac{6.12}{0.22}\]
\[t_{obt} = {27.82}\]

calculate Cohen’s d
\[d = \frac{M - \mu}{s}\]
\[d = \frac{19.12 - 13}{0.64}\]
\[d = \frac{6.12}{0.64}\]
\[d = {9.56}\]

The results:

reject the null hypothesis, results are significant,
t(7) = 27.82, p < 0.05, d = 9.56
