Problem 1

Given the following data, is the mean of \(Group_A\) significantly different from \({44}\)? Use a \({2}\) tailed-test with \(\alpha = {0.05}\)

A
61
58
54
46
58
67
53
71
54
37
61
56
62
58


The necessary summary statistics for these data
\[M_A = {56.86}\] \[s^2 = {70.13}\] \[n = {14}\]

State the Hypotheses
\[H_0: \mu = 44\] \[H_1: \mu \ne 44\]


The decision criteria:

\(t_{crit} = \pm{2.16}, \alpha_{two-tailed} = {0.05}, df = {13}\)

Calculate the standard error
\[s_M = \sqrt{\frac{s^2}{n}}\]
\[s_M = \sqrt{\frac{70.13}{14}}\]
\[s_M = \sqrt{5.01}\]
\[s_M = {2.24}\]

calculate \(t_{obt}\)
\[t_{obt} = {\frac{M - \mu}{s_M}}\]
\[t_{obt} = \frac{56.86 - 44}{2.24}\]
\[t_{obt} = \frac{12.86}{2.24}\]
\[t_{obt} = {5.74}\]

calculate Cohen’s d
\[d = \frac{M - \mu}{s}\]
\[d = \frac{56.86 - 44}{8.37}\]
\[d = \frac{12.86}{8.37}\]
\[d = {1.54}\]

The results:

reject the null hypothesis, results are significant,
t(13) = 5.74, p < 0.05, d = 1.54
