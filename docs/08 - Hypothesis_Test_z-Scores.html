Problem 1

Given the following data, is the mean of \(Group_A\) significantly different from the population mean: \(\mu = {21}\)?

Use a \({2}\) tailed-test with \(\alpha = {0.01}\)

A
38
33
31
30
34
29
31
31


The necessary summary statistics for these data
\[M_A = {32.12}\] \[{\sigma} = {4}\] \[n = {8}\]

State the Hypotheses
\[H_0: \mu = 21\] \[H_1: \mu \ne 21\]


The decision criteria:

\(z_{crit} = \pm{2.58}, \alpha_{two-tailed} = {0.01}\)

Calculate the standard error
\[\sigma_M = \frac{\sigma}{\sqrt{N}}\]
\[\sigma_M = \frac{4}{\sqrt{8}}\]
\[\sigma_M = \frac{4}{2.83}\]
\[\sigma_M = {1.41}\]

Calculate \(z_{obt}\)
\[z_{obt} = {\frac{M - \mu}{\sigma_M}}\]
\[z_{obt} = \frac{32.12 - 21}{1.41}\]
\[z_{obt} = \frac{11.12}{1.41}\]
\[z_{obt} = {7.89}\]

Calculate Cohen’s d for effect size
\[d = \frac{M - \mu}{\sigma}\]
\[d = \frac{32.12 - 21}{4}\]
\[d = \frac{11.12}{4}\]
\[d = {2.78}\]

The results:

reject the null hypothesis, results are significant,
z = 7.89, p < 0.01, d = 2.78
