Problem 1

Given the following data, is the mean of \(Group_A\) significantly different from the population mean: \(\mu = {28}\)?

Use a \({2}\) tailed-test with \(\alpha = {0.01}\)

A
45
38
42
35
42
35


The necessary summary statistics for these data
\[M_A = {39.5}\] \[{\sigma} = {4}\] \[n = {6}\]

State the Hypotheses
\[H_0: \mu = 28\] \[H_1: \mu \ne 28\]


The decision criteria:

\(z_{crit} = \pm{2.58}, \alpha_{two-tailed} = {0.01}\)

Calculate the standard error
\[\sigma_M = \frac{\sigma}{\sqrt{N}}\]
\[\sigma_M = \frac{4}{\sqrt{6}}\]
\[\sigma_M = \frac{4}{2.45}\]
\[\sigma_M = {1.63}\]

Calculate \(z_{obt}\)
\[z_{obt} = {\frac{M - \mu}{\sigma_M}}\]
\[z_{obt} = \frac{39.5 - 28}{1.63}\]
\[z_{obt} = \frac{11.5}{1.63}\]
\[z_{obt} = {7.06}\]

Calculate Cohen’s d for effect size
\[d = \frac{M - \mu}{\sigma}\]
\[d = \frac{39.5 - 28}{4}\]
\[d = \frac{11.5}{4}\]
\[d = {2.88}\]

The results:

reject the null hypothesis, results are significant,
z = 7.06, p < 0.01, d = 2.88
