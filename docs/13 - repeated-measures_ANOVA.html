Summary statistics for these data:
\(G = {1956} \quad \Sigma X^2 = {56758} \quad k = {5} \quad N = {70}\)

\(T_{A} = {388} \quad SS_{A} = {166.86}\)
\(T_{B} = {502} \quad SS_{B} = {183.71}\)
\(T_{C} = {336} \quad SS_{C} = {86.0}\)
\(T_{D} = {326} \quad SS_{D} = {126.86}\)
\(T_{E} = {404} \quad SS_{E} = {127.71}\)

State the Hypotheses
\(H_0: \mu_A = \mu_B = \mu_C = \mu_D = \mu_E\)
\(H_1:\) At least one mean is different

The decision criteria:

\(F_{crit} = {2.55}, \alpha = {0.05}\)

Stage 1 Calculations:

Calculate the Degrees of Freedom
\[df_{total} = N - 1\]
\[df_{total} = {70} - 1\]
\[df_{total} = {69}\]

\[df_{between} = k - 1\]
\[df_{between} = {5} - 1\]
\[df_{between} = {4}\]

\[df_{within} = N - K\]
\[df_{within} = {70} - {5}\]
\[df_{within} = {65}\]

Calculate the Sum of Squares
\[SS_{total} = \Sigma X^2 - \frac{G^2}{N}\]
\[SS_{total} = {56758} - \frac{1956^2}{70}\]
\[SS_{total} = {56758} - \frac{3825936}{70}\]
\[SS_{total} = {56758} - {54656.23}\]
\[SS_{total} = {2101.77}\]

\[SS_{within} = \Sigma SS_{inside\_each\_condition}\]
\[SS_{within} = {166.86 + 183.71 + 86.0 + 126.86 + 127.71}\]
\[SS_{within} = {691.14}\]

\[SS_{between} = SS_{total} - SS_{within}\]
\[SS_{between} = {2101.77} - {691.14}\]
\[SS_{between} = {1410.63}\]

note: the other way to calculate \(SS_{betwen}\) is:
\[SS_{between} = \Sigma{\frac{T^2}{n}} - \frac{G^2}{N}\]

Stage 2 Calculations:

Partition the Degrees of Freedom
\[df_{subjects} = n - 1\]
\[df_{subjects} = {14} - 1\]
\[df_{subjects} = {13}\]

\[df_{error} = df_{within} - df_{subjects}\]
\[df_{error} = {65} - {13}\]
\[df_{error} = {52}\]

Calculate person sums and add a new column (P)

\[SS_{error} = SS_{within} - SS_{subjects}\]
\[SS_{error} = {691.14} - {161.37}\]
\[SS_{error} = {529.77}\]

Calculate the Mean Squares
\[MS_{between} = \frac{SS_{between}}{df_{between}}\]
\[MS_{between} = \frac{1410.63}{4}\]
\[MS_{between} = {352.66}\]

\[MS_{error} = \frac{SS_{error}}{df_{error}}\]
\[MS_{error} = \frac{529.77}{52}\]
\[MS_{error} = {10.19}\]

Calculate the F-Ratio
\[F_{obt} = \frac{MS_{between}}{MS_{error}}\]
\[F_{obt} = \frac{352.66}{10.19}\]
\[F_{obt} = {34.61}\]

Calculate \(\eta^2\)
\[\eta_p^2 = \frac{SS{between}}{SS_{total} - SS_{subjects}}\]
\[\eta_p^2 = \frac{1410.63}{{2101.77} - {161.37}}\]
\[\eta_p^2 = \frac{1410.63}{1940.4}\]
