Hypothesis Testing with z-scores

Given the following data, is the mean of \(Group_A\) significantly different from the population mean: \(\mu = {41}\)?

Use a \({2}\) tailed-test with \(\alpha = {0.05}\)

A
47
42
42
48
38
46
46
35
43
41
48


The necessary summary statistics for these data
\[M_A = {43.27}\] \[{\sigma} = {8}\] \[n = {11}\]

State the Hypotheses
\[H_0: \mu = 41\] \[H_1: \mu \ne 41\]


The decision criteria:

\(z_{crit} = \pm{1.96}, \alpha_{two-tailed} = {0.05}\)

Calculate the standard error
\[\sigma_M = \frac{\sigma}{\sqrt{N}}\]
\[\sigma_M = \frac{8}{\sqrt{11}}\]
\[\sigma_M = \frac{8}{3.32}\]
\[\sigma_M = {2.41}\]

Calculate \(z_{obt}\)
\[z_{obt} = {\frac{M - \mu}{\sigma_M}}\]

\[d = \frac{M - \mu}{\sigma}\]
\[d = \frac{43.27 - 41}{8}\]
\[d = \frac{2.27}{8}\]
\[d = {0.28}\]

The results:

fail to reject the null hypothesis, results not significant,
z = 0.94, p > 0.05, d = 0.28

Independent-Samples t-test

Given the following between-subjects data, is the mean of \(Group_A\) significantly different from the mean of \(Group_B\)? Use a \({2}\) tailed-test with \(\alpha = {0.05}\)

A B
24 17
35 28
30 24
28 22
31 23
33 26


The necessary summary Statistis for these data:
\[M_A = {30.17}, M_B = {23.33}\] \[SS_A = {74.83}, SS_B = {71.33}\] \[n_A = {6}, n_B = {6}\]

State the Hypotheses
\[H_0: \mu_A - \mu_B = 0\] \[H_1: \mu_A - \mu_B \ne 0\]


The decision criteria:

\(t_{crit} = \pm{2.23}, \alpha_{two-tailed} = {0.05}, df = {10}\)

Calculate the pooled variance:
\[s_p^2 = {\frac{SS_A + SS_B}{df_A + df_B}}\]
\[s_p^2 = {\frac{74.83 + 71.33}{5 + 5}}\]
\[s_p^2 = \frac{146.16}{10}\]
\[s_p^2 = {14.62}\]

Calculate the estimated error of the difference between means: \(s_{(M_A - M_B)}\)
\[s_{(M_A - M_B)} = \sqrt{\frac{s_p^2}{n_1} + \frac{s_p^2}{n_1}}\]
\[s_{(M_A - M_B)} = \sqrt{\frac{14.62}{6} + \frac{14.62}{6}}\]
\[s_{(M_A - M_B)} = \sqrt{2.44 + 2.44}\]
\[s_{(M_A - M_B)} = \sqrt{4.88}\]
\[s_{(M_A - M_B)} = {2.21}\]

Calculate \(t_{obt}\)
\[t_{obt} = {\frac{(M_A - M_B) - (\mu_A - \mu_B)}{s_{(M_A - M_B)}}}\]
\[t_{obt} = \frac{(30.17 - 23.33) - {0}}{2.21}\]
\[t_{obt} = \frac{6.84}{2.21}\]
\[t_{obt} = {3.1}\]

Calculate Cohen’s d
\[d = \frac{M_A - M_B}{\sqrt{s_p^2}}\]
\[d = \frac{(30.17 - 23.33)}{{{\sqrt{14.62}}}}\]
\[d = \frac{(6.84)}{3.82}\]
\[d = {1.79}\]

The results:

reject the null hypothesis, results are significant,
t(10) = 3.1, p < 0.05, d = 1.79

\(G = {2046} \quad \Sigma{X^2} = {79392} \quad k = {5} \quad N = {55}\)

\(T_{A} = {478} \quad SS_{A} = {514.73}\)
\(T_{B} = {351} \quad SS_{B} = {320.91}\)
\(T_{C} = {367} \quad SS_{C} = {372.55}\)
\(T_{D} = {400} \quad SS_{D} = {372.55}\)

State the Hypotheses
\(H_0: \mu_A = \mu_B = \mu_C = \mu_D = \mu_E\)

\(F_{crit} = {3.72}, \alpha = {0.01}\)

\[df_{total} = {55} - 1\]
\[df_{total} = {54}\]

\[df_{within} = N - K\]
\[df_{within} = {55} - {5}\]
\[df_{within} = {50}\]

\[SS_{total} = \Sigma X^2 - \frac{G^2}{N}\]
\[SS_{total} = {79392} - \frac{2046^2}{55}\]
\[SS_{total} = {79392} - \frac{4186116}{55}\]
\[SS_{total} = {79392} - {76111.2}\]
\[SS_{total} = {3280.8}\]

\[SS_{within} = {514.73 + 320.91 + 372.55 + 372.55 + 640.91}\]

\[F_{obt} = \frac{MS_{between}}{MS_{within}}\]
\[F_{obt} = \frac{264.79}{44.43}\]
\[F_{obt} = {5.96}\]

\[\eta^2 = \frac{SS{between}}{SS_{total}}\]
\[\eta^2 = \frac{1059.15}{3280.8}\]
\[\eta^2 = {0.32}\]

The results:

reject the null hypothesis, results are significant,

\(F({4}, {50}) = {5.96}, p < {0.01}, \eta^2 = {0.32}\)
