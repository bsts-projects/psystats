Hypothesis Testing with z-scores

Given the following data, is the mean of \(Group_A\) significantly different from the population mean: \(\mu = {29}\)?

Use a \({2}\) tailed-test with \(\alpha = {0.05}\)

A

The necessary summary statistics for these data
\[M_A = {40.22}\] \[{\sigma} = {8}\] \[n = {9}\]

State the Hypotheses
\[H_0: \mu = 29\] \[H_1: \mu \ne 29\]


The decision criteria:

\(z_{crit} = \pm{1.96}, \alpha_{two-tailed} = {0.05}\)

Calculate the standard error
\[\sigma_M = \frac{\sigma}{\sqrt{N}}\]
\[\sigma_M = \frac{8}{\sqrt{9}}\]
\[\sigma_M = \frac{8}{3.0}\]
\[\sigma_M = {2.67}\]

Calculate \(z_{obt}\)

\[d = {1.4}\]

The results:

reject the null hypothesis, results are significant,
z = 4.2, p < 0.05, d = 1.4

Independent-Samples t-test

Given the following between-subjects data, is the mean of \(Group_A\) significantly different from the mean of \(Group_B\)? Use a \({2}\) tailed-test with \(\alpha = {0.05}\)

A B
23 17
25 26
28 28
23 16
23 19


The necessary summary Statistis for these data:
\[M_A = {24.4}, M_B = {21.2}\] \[SS_A = {19.2}, SS_B = {118.8}\] \[n_A = {5}, n_B = {5}\]

State the Hypotheses
\[H_0: \mu_A - \mu_B = 0\] \[H_1: \mu_A - \mu_B \ne 0\]


The decision criteria:

\(t_{crit} = \pm{2.31}, \alpha_{two-tailed} = {0.05}, df = {8}\)

Calculate the pooled variance:
\[s_p^2 = {\frac{SS_A + SS_B}{df_A + df_B}}\]
\[s_p^2 = {\frac{19.2 + 118.8}{4 + 4}}\]
\[s_p^2 = \frac{138.0}{8}\]
\[s_p^2 = {17.25}\]

Calculate the estimated error of the difference between means: \(s_{(M_A - M_B)}\)
\[s_{(M_A - M_B)} = \sqrt{\frac{s_p^2}{n_1} + \frac{s_p^2}{n_1}}\]
\[s_{(M_A - M_B)} = \sqrt{\frac{17.25}{5} + \frac{17.25}{5}}\]
\[s_{(M_A - M_B)} = \sqrt{3.45 + 3.45}\]
\[s_{(M_A - M_B)} = \sqrt{6.9}\]
\[s_{(M_A - M_B)} = {2.63}\]

Calculate \(t_{obt}\)
\[t_{obt} = {\frac{(M_A - M_B) - (\mu_A - \mu_B)}{s_{(M_A - M_B)}}}\]
\[t_{obt} = \frac{(24.4 - 21.2) - {0}}{2.63}\]
\[t_{obt} = \frac{3.2}{2.63}\]
\[t_{obt} = {1.22}\]

Calculate Cohen’s d
\[d = \frac{M_A - M_B}{\sqrt{s_p^2}}\]
\[d = \frac{(24.4 - 21.2)}{{{\sqrt{17.25}}}}\]
\[d = \frac{(3.2)}{4.15}\]
\[d = {0.77}\]

The results:

fail to reject the null hypothesis, results not significant,
t(8) = 1.22, p > 0.05, d = 0.77

One-Way ANOVA

Given the following between-subjects data, use a one-way ANOVA with \(\alpha = {0.05}\)

A B C D E
29 39 18 18 30
25 45 22 23 32
27 39 23 24 37
36 39 21 27 20
37 41 19 29 28


Summary statistics for these data:
\(G = {728} \quad \Sigma{X^2} = {22744} \quad k = {5} \quad N = {25}\)

\(T_{A} = {154} \quad SS_{A} = {116.8}\)
\(T_{B} = {203} \quad SS_{B} = {27.2}\)
\(T_{C} = {103} \quad SS_{C} = {17.2}\)
\(T_{D} = {121} \quad SS_{D} = {70.8}\)
\(T_{E} = {147} \quad SS_{E} = {155.2}\)

State the Hypotheses
\(H_0: \mu_A = \mu_B = \mu_C = \mu_D = \mu_E\)
\(H_1:\) At least one mean is different

The decision criteria:

\(F_{crit} = {2.87}, \alpha = {0.05}\)

Calculate the Degrees of Freedom
\[df_{total} = N - 1\]
\[df_{total} = {25} - 1\]
\[df_{total} = {24}\]

\[df_{between} = k - 1\]
\[df_{between} = {5} - 1\]
\[df_{between} = {4}\]

\[df_{within} = N - K\]
\[df_{within} = {25} - {5}\]
\[df_{within} = {20}\]

Calculate the Sum of Squares
\[SS_{total} = \Sigma X^2 - \frac{G^2}{N}\]
\[SS_{total} = {22744} - \frac{728^2}{25}\]
\[SS_{total} = {22744} - \frac{529984}{25}\]
\[SS_{total} = {22744} - {21199.36}\]
\[SS_{total} = {1544.64}\]

\[SS_{within} = \Sigma SS_{inside\_each\_condition}\]
\[SS_{within} = {116.8 + 27.2 + 17.2 + 70.8 + 155.2}\]
\[SS_{within} = {387.2}\]

\[SS_{between} = SS_{total} - SS_{within}\]
\[SS_{between} = {1544.64} - {387.2}\]
\[SS_{between} = {1157.44}\]

note: the other way to calculate \(SS_{betwen}\) is:
\[SS_{between} = \Sigma{\frac{T^2}{n}} - \frac{G^2}{N}\]

Calculate the Mean Squares
\[MS_{between} = \frac{SS_{between}}{df_{between}}\]
\[MS_{between} = \frac{1157.44}{4}\]
\[MS_{between} = {289.36}\]

\[MS_{within} = \frac{SS_{within}}{df_{within}}\]
\[MS_{within} = \frac{387.2}{20}\]
\[MS_{within} = {19.36}\]

Calculate the F-Ratio
\[F_{obt} = \frac{MS_{between}}{MS_{within}}\]
\[F_{obt} = \frac{289.36}{19.36}\]
\[F_{obt} = {14.95}\]

Calculate \(\eta^2\)
\[\eta^2 = \frac{SS{between}}{SS_{total}}\]
\[\eta^2 = \frac{1157.44}{1544.64}\]
\[\eta^2 = {0.75}\]

The results:

reject the null hypothesis, results are significant,

\(F({4}, {20}) = {14.95}, p < {0.05}, \eta^2 = {0.75}\)
