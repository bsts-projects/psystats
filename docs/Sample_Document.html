Hypothesis Testing with z-scores

Given the following data, is the mean of \(Group_A\) significantly different from the population mean: \(\mu = {32}\)?

Use a \({2}\) tailed-test with \(\alpha = {0.01}\)

A
27
41
48

27

The necessary summary statistics for these data
\[M_A = {37.5}\] \[{\sigma} = {8}\] \[n = {10}\]

State the Hypotheses
\[H_0: \mu = 32\] \[H_1: \mu \ne 32\]


The decision criteria:

\(z_{crit} = \pm{2.58}, \alpha_{two-tailed} = {0.01}\)

Calculate the standard error
\[\sigma_M = \frac{\sigma}{\sqrt{N}}\]
\[\sigma_M = \frac{8}{\sqrt{10}}\]
\[\sigma_M = \frac{8}{3.16}\]
\[\sigma_M = {2.53}\]

Calculate \(z_{obt}\)

\[d = \frac{37.5 - 32}{8}\]
\[d = \frac{5.5}{8}\]
\[d = {0.69}\]

The results:

fail to reject the null hypothesis, results not significant,
z = 2.17, p > 0.01, d = 0.69

Independent-Samples t-test

Given the following between-subjects data, is the mean of \(Group_A\) significantly different from the mean of \(Group_B\)? Use a \({2}\) tailed-test with \(\alpha = {0.01}\)

A B
39 47
30 62
38 50

41 54

The necessary summary Statistis for these data:
\[M_A = {36.56}, M_B = {55.67}\] \[SS_A = {102.22}, SS_B = {180.0}\] \[n_A = {9}, n_B = {9}\]

State the Hypotheses
\[H_0: \mu_A - \mu_B = 0\] \[H_1: \mu_A - \mu_B \ne 0\]


The decision criteria:

\(t_{crit} = \pm{2.92}, \alpha_{two-tailed} = {0.01}, df = {16}\)

Calculate the pooled variance:
\[s_p^2 = {\frac{SS_A + SS_B}{df_A + df_B}}\]
\[s_p^2 = {\frac{102.22 + 180.0}{8 + 8}}\]
\[s_p^2 = \frac{282.22}{16}\]
\[s_p^2 = {17.64}\]

Calculate the estimated error of the difference between means: \(s_{(M_A - M_B)}\)
\[s_{(M_A - M_B)} = \sqrt{\frac{s_p^2}{n_1} + \frac{s_p^2}{n_1}}\]
\[s_{(M_A - M_B)} = \sqrt{\frac{17.64}{9} + \frac{17.64}{9}}\]
\[s_{(M_A - M_B)} = \sqrt{1.96 + 1.96}\]
\[s_{(M_A - M_B)} = \sqrt{3.92}\]
\[s_{(M_A - M_B)} = {1.98}\]

Calculate \(t_{obt}\)
\[t_{obt} = {\frac{(M_A - M_B) - (\mu_A - \mu_B)}{s_{(M_A - M_B)}}}\]
\[t_{obt} = \frac{(36.56 - 55.67) - {0}}{1.98}\]
\[t_{obt} = \frac{-19.11}{1.98}\]
\[t_{obt} = {-9.65}\]

Calculate Cohen’s d
\[d = \frac{M_A - M_B}{\sqrt{s_p^2}}\]
\[d = \frac{(36.56 - 55.67)}{{{\sqrt{17.64}}}}\]
\[d = \frac{(-19.11)}{4.2}\]
\[d = {-4.55}\]

The results:

reject the null hypothesis, results are significant,
t(16) = -9.65, p < 0.01, d = -4.55
